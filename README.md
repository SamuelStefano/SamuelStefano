## 🛰️ Deck — my own AI cockpit

<div align="center">

> A personal **command center for AI agents**, running on my own VPS — chat with Claude side-by-side with live tmux terminals, persistent memory, skills, model auto-routing, token budgeting and host administration, all in one web interface I can open from any device.

</div>

<table>
  <tr>
    <td valign="top" width="50%">

**What Deck is**
- 💬 **Multi-agent chat** with Claude (Opus / Sonnet / Haiku) and open models via OpenRouter
- 🖥️ **Live terminals** through tmux on the VPS — sessions survive reboots and disconnects
- 🧠 **Persistent memory** across sessions, with auto-saved user / project / feedback notes
- 🛠️ **Skills marketplace** — modular agent capabilities I can install per-project
- 💸 **Token budgeting** and cost telemetry per task (Pro plan + API key separation)
- 🛡️ **Host administration** — supervisor, doctor scripts, idle session hibernation, security gates

    </td>
    <td valign="top" width="50%">

**Why it exists**
- I wanted to ship work **from my phone while walking around** — not be glued to a desktop
- Every agent runs in **my** environment, with **my** credentials, **my** vault, **my** rules
- Local-first, VPS-first, zero third-party clouds in the loop
- A single pane of glass for everything I'm building across DFL + personal projects
